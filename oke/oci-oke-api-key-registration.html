OKE API Key 등록

OCI Console을 통해 API Key를 자동 생성하여 등록할 수 있으며, 이는 OCI CLI 및 SDK를 통해 접속하는 OCI RESTful API Endpoint에 대한 기본 인증으로 활용됩니다. API Key는 OCI Kubernetes Cluster 인증에도 사용됩니다.

1) User Setting 페이지로 이동​

OCI Console의 우측 상단에 사용자 아이콘을 클릭하여 User Settings을 선택합니다.

2) API Key 생성

왼쪽 하단의 Resources 메뉴에서 API Keys를 선택한 후, Add API Key를 클릭합니다. ✔️

3) API Key 다운 로드 및 등록

API Key Pair(Public/Private Keys)는 OCI Console을 통해 자동으로 생성할 수 있습니다.

  • Private Key와 Public Key를 클릭하여 다운로드 후, 안전한 위치에 보관합니다.

    • Public Key : OCI에 보관되는 API Key Pair
    • Private Key : OCI로의 인증을 위해 Client에서 사용할 API Key Pair

주의: 본 페이지 이후에는 별도의 다운로드가 불가능합니다.

4) Configuration File 리뷰

  • Client에 설정할 Configuration File의 내용을 미리 확인해볼 수 있습니다.
  • 이는 복사하여 SDK/CLI 연결용 Configuration File로 활용 가능합니다.

Linux의 경우, ~/.oci/config

5) 등록된 API Key 확인

등록된 API Key에 대한 Configuration File의 내용은 View Configuration file 메뉴를 통해 언제든지 다시 참조가 가능합니다.
